Column NameValue
Script Namegeosquare_crawler.py
OverviewRecursively crawls web pages from a given starting URL, extracting and saving links.
PrerequisitesPython 3.x, Selenium, Pandas, JSON, WebDriverManager.
Installationpip install selenium pandas webdriver-manager
Global VariablesBase URL, working directory for saving state and results, Selenium WebDriver setup.
Functionssetup_driver (configures WebDriver), extract_links (retrieves all links from a page), recursive_crawl (performs depth-first link extraction).
Error HandlingHandles page load detection, retries failed extractions, saves state to resume crawling.
OutputExtracted links are saved in CSV files, discovered links in JSON, and state in a checkpoint file.
ExecutionRun using `python geosquare_crawler.py`.
NotesResumes from last saved state, avoids duplicate crawling, logs visited URLs.
