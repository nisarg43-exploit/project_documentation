Column NameValue
Script NameMaharera_download_doc.py, maharera_project_details_download.py
OverviewScrapes and downloads project details and documents from the MahaRERA website.
PrerequisitesPython 3.x, Selenium, OpenCV, Requests, Google Cloud Storage, Pandas, Pytesseract.
Installationpip install selenium opencv-python requests google-cloud-storage pandas pytesseract
Global VariablesGoogle Cloud Storage credentials, project registration numbers, captcha OCR model.
Functionsget_proxy_driver (sets up ChromeDriver), load_website (handles retries), process_captcha (solves captchas), main_scrap (fetches project data and documents).
Error HandlingHandles CAPTCHA solving using OCR, retries failed downloads, logs errors in processing.
OutputExtracted project details saved in structured CSV format, PDFs and images uploaded to cloud.
ExecutionRun using `python Maharera_download_doc.py` or `python maharera_project_details_download.py`.
NotesRuns in a loop, processes multiple project registration numbers from MahaRERA.
