Script Namevoters_data_download.py
OverviewAutomates the downloading of voter data PDFs from the Election Commission of India website.
PrerequisitesPython 3.x, Selenium, OpenCV, Requests, Google Cloud Storage, Pandas.
Installationpip install selenium opencv-python requests google-cloud-storage pandas
Global VariablesGoogle Cloud Storage credentials, input district list, captcha OCR model.
Functionsget_proxy_driver (sets up ChromeDriver), main_scrap (fetches and downloads PDFs), foldercheck (manages file directories).
Error HandlingHandles CAPTCHA solving using OCR, retries on failures, logs errors in processing.
OutputExtracted voter PDFs saved locally and uploaded to Google Cloud Storage.
ExecutionRun using `python voters_data_download.py`.
NotesRuns in a loop, processes multiple districts and constituencies.
