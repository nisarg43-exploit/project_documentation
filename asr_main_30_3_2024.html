Script Nameasr_main_30_3_2024.py
OverviewScrapes assessment-related data from the Maharashtra government website using Selenium.
PrerequisitesPython 3.x, Google Chrome, ChromeDriver, Selenium, pandas, requests, google-cloud-storage, OpenCV.
Installationpip install selenium pandas requests google-cloud-storage opencv-python
Global VariablesDictionary mapping numeric values to year ranges, district name, year selection, CSV filename, headless mode.
Functionsget_proxy_driver (sets up ChromeDriver), checkloading (waits for page load), main_scrap (performs scraping).
Error HandlingRetries element selection on failure, handles exceptions, logs errors in `{district_name}_error_villages.txt`.
OutputExtracted data saved in `{year_dist[year_value]}_{district_name}.csv`, completed villages logged in `{district_name}_completed_villages.txt`.
ExecutionRun using `python asr_main_30_3_2024.py`.
NotesRuns in an infinite loop, can be modified for other districts and years.
