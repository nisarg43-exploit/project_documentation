Script Namebhunaksha_plot_data_scrap.py, bhunaksha_map_image_scrap.py
OverviewExtracts plot data and cadastral maps from the Bhunaksha website.
PrerequisitesPython 3.x, Selenium, Requests, Pandas, Google Cloud Storage.
Installationpip install selenium opencv-python requests google-cloud-storage pandas
Global VariablesGoogle Cloud Storage credentials, input CSV file with plot details.
Functionsprocess_plot_info (fetches plot details), url_modify (adjusts image URLs), main_scrap (fetches images).
Error HandlingHandles failed requests, missing GIS codes, retries on errors, logs unprocessed plots.
OutputExtracted plot data saved in CSV, images uploaded to Google Cloud.
ExecutionRun using `python bhunaksha_plot_data_scrap.py` or `python bhunaksha_map_image_scrap.py`.
NotesContinuously runs, processes images and plot data dynamically.
