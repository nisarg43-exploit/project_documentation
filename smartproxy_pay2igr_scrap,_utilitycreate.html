Script Namesmartproxy_pay2igr_scrap.py, utilitycreate.py
OverviewAutomates the scraping of property records from the IGR Maharashtra website using SmartProxy.
PrerequisitesPython 3.x, Selenium, Pandas, Google Cloud Storage, BeautifulSoup, JSON.
Installationpip install selenium pandas google-cloud-storage beautifulsoup4
Global VariablesSmartProxy credentials, Google Cloud Storage credentials, district and taluka mapping JSON files.
Functionsget_proxy_driver (configures SmartProxy for web scraping), webdriver_example (launches browser), main_scrap (performs data extraction and storage).
Error HandlingHandles CAPTCHA solving, retries failed requests, logs errors in processing.
OutputExtracted property records saved in structured CSV format, uploaded to Google Cloud Storage.
ExecutionRun using `python smartproxy_pay2igr_scrap.py` after configuring settings via `utilitycreate.py`.
NotesWorks with district and taluka mappings to dynamically generate scraping tasks.
